What are Private Cloud Instances?

ThingsIoT Private instances are isolated servers for each customer, so the instance is not shared with other thousands of users from our community. Private instances can be started in different cloud providers like DigitalOcean, Amazon Web Services, Google Cloud, or Azure in different availability zones .

Does ThingsIoT Support On-Permise Installs?

Yes, ThingsIoT supports on-premise installs over Docker by using a License Key to enable certain platform features.

Is it Possible to Deploy Highly Available ThingsIoT Cloud Cluster?

Yes, for large and critical deployments it is possible to launch ThingsIoT in a cluster. Please, contact us for a quote.

What are Benefits of Private Cloud Instance Instead of Free Public Instance?

Private instances has several benefits over our community edition, like the possibility to install plugins; define custom domains; rebrand the solution; manage instance users, or support databases with real-time data aggregation for dashboards. Moreover, as it is an isolated instance, it has a predictable performance and throughput that can be scaled as required.
